## Notes for Developers

- **Tech stack highlights:**
  - Flutter for the main app (`falcon-app`)  
  - Self Hosted database - PostgreSQL
  - Hive for offline storage  
  - GoRouter for navigation  

- **Workflow tips:**
  - `falcon-docs` and `falcon-website` are static site projects.  
  - `falcon-app` is Flutter multi-platform

---

## Goals

- Keep Falcon Logbook **open-source, transparent, and community-driven**  
- Make the **best pilot logbook experience** without locking pilots into expensive subscriptions  
- Maintain **lean infrastructure and privacy-first principles**
